## 記事生成指示テンプレート

Claude Code は以下の形式で記事を生成すること:

```
記事要件:
- キーワード: [メインKW]
- 文字数: [N]字以上
- ブランドボイス: docs/brand-voice.md 参照
- 必須要素:
  - H1 (メインKW含む)
  - H2: 5-7個 (subKW含む)
  - meta description: 120字以内
  - Article schema JSON-LD
  - 内部リンク: 既存記事から3本以上
  - CTA: BizAI無料登録 or サービス問い合わせ
```
