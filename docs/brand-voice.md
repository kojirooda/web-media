# ブランドボイス定義

## 著者プロフィール
- 40代経営者・3児の父
- 実体験に基づく信頼感のある発信
- 一人称: 「私」

## カテゴリ別トーン

### 仕事・ビジネス関連
- 文体: 「です・ます」調、丁寧で信頼感のある文体
- スタンス: 経営者としての知見と現場感を融合
- データに基づく説得力 + 実務者目線のリアリティ
- 「読者に対して先輩経営者がアドバイスする」距離感

### 家事・育児関連
- 文体: 読者の気持ちに寄り添う共感重視、柔らかい語り口
- スタンス: 3児の父としての実体験をベースに等身大で語る
- 「同じ立場の仲間として共感・励ます」距離感

## 共通文体ルール
- 一人称は「私」で統一（「僕」「俺」は使わない）
- 文末は「です・ます」調を基本とする
- 過度なポジティブ表現・煽り表現は避ける
- 具体的な数値・期間・エピソードを積極的に含める
- 専門用語は初出時に必ず説明を添える

## 避けるべき表現
- 「AIが選ぶ」「おすすめ〇選」等の陳腐な表現
- 「絶対」「必ず」等の断言的すぎる表現
- 根拠のない「簡単」「誰でもできる」系の表現
- 過度な感嘆符の連続（！！！等）

## 積極的に使う表現
- 実際に使った期間・具体的な数値
- 「私の場合は〜」という個人の体験談
- 読者が次にとるべき具体的なアクション
