# Audience Profile スキル

すべてのコンテンツ生成タスクで、以下のペルソナを参照してください。

## メインターゲット

**年齢・属性**: 20代〜40代の男女

### ペルソナA：ビジネス・仕事系コンテンツの読者
- **年齢**: 30〜40代
- **状況**: 会社員または独立・副業を検討中
- **主な悩み**: キャリアアップ、生産性向上、副業・起業、マネジメント
- **検索行動**: 「〇〇 方法」「〇〇 やり方」「〇〇 コツ」等の How-to 検索が多い
- **期待値**: 実際に試した人の体験談、すぐ実践できる具体的な手順

### ペルソナB：家事・育児系コンテンツの読者
- **年齢**: 20〜40代
- **状況**: 子育て中の親（特に共働き世帯）
- **主な悩み**: 時短・家事効率化、子どもの教育・発達、夫婦間の役割分担
- **検索行動**: 「〇〇 簡単」「〇〇 子ども」「〇〇 共働き」等
- **期待値**: 同じ立場の人の共感、実際に効果があった方法、失敗談も含めたリアルな情報

## コンテンツ制作時の必須確認事項

1. **検索意図の分類**: 情報収集型 / 比較検討型 / 解決策探索型 のどれか
2. **読者の知識レベル**: 初心者向けか経験者向けかを明示する
3. **読者が記事を読んだ後に取るべきアクション**を必ず含める
4. **読者の悩みを冒頭で言語化**して共感を得てから解決策へ進む

## NG コンテンツパターン

- ターゲットの悩みとずれた内容（読者目線がない情報羅列）
- 専門家でないと理解できない専門用語の多用
- 「〜すべき」「〜しなければならない」等の圧迫感のある表現
