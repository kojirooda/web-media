記事執筆パイプラインを実行します。以下のステップを順番に実施してください。

## 使い方
```
/write [テーマ or キーワード]
```

---

## Step 1: キーワードリサーチ・構成案作成

1. 指定テーマで WebSearch を実行（検索意図・競合・関連KWを調査）
2. メインキーワード1つ・サブキーワード3つを決定
3. `content/drafts/[slug]/outline.md` を作成して保存

**outline.md に含める項目:**
- 基本情報（slug, カテゴリ, KW, meta description 120文字以内）
- H1 タイトル
- H2/H3 見出し構造
- 各セクションの要点メモ

---

## Step 2: 本文執筆

`brand-voice` スキルと `audience-profile` スキルを適用して本文を執筆します。

**必須チェック項目:**
- [ ] 一人称「私」で統一
- [ ] 「です・ます」調
- [ ] 具体的な数値・期間・体験談を含む
- [ ] 導入部で読者の悩みを言語化
- [ ] まとめに具体的な次のアクション（CTA）
- [ ] 関連記事リンクを3本以上含む

完成した本文を `content/drafts/[slug]/article.md` に保存してください。

---

## Step 3: アイキャッチ画像プロンプト生成【必須】

記事の内容を踏まえ、Antigravity（Gemini Pro）用の画像生成プロンプトを作成し、
`content/drafts/[slug]/image-prompt.md` に保存してください。

**プロンプトに含める要素:**
- 記事テーマを視覚的に表現するシーン設定
- 雰囲気・トーン（プロフェッショナル / 温かみ / 誠実 など）
- 推奨カラーパレット
- テキスト配置を考慮した構図指定
- サイズ指定: **1200×630px（OGP横型）**
- ネガティブプロンプト（避けるべき要素）

**image-prompt.md のフォーマット:**
```markdown
# アイキャッチ画像プロンプト

## 記事タイトル
[タイトル]

## Antigravity 用プロンプト（日本語）
[詳細なプロンプト]

## Antigravity 用プロンプト（英語）
[詳細なプロンプト - 英語の方が精度が高い場合に使用]

## ネガティブプロンプト
[避けるべき要素]

## サイズ・仕様
- サイズ: 1200×630px
- 用途: アイキャッチ（OGP）
- 保存先: public/images/[slug]/eyecatch.png
```

---

## Step 4: QA チェック

`/review` を実行して `creative-qa` スキルによる品質チェックを行います。
Critical が 0 件になるまで修正してください。

---

## Step 5: Notion 編集カレンダーへ登録

Notion MCP を使って編集カレンダー DB に以下を登録してください:
- タイトル
- ステータス: レビュー
- 担当エージェント: Claude Code
- キーワード
- 優先度
- 公開予定日（本日から7日後を目安）

---

## 完了報告フォーマット

```
## /write 完了レポート
- 記事タイトル: [タイトル]
- スラッグ: [slug]
- 文字数: 約[X]文字
- QA結果: Critical [X]件 / Important [X]件
- 画像プロンプト: content/drafts/[slug]/image-prompt.md に保存済み
- Notion登録: ✅ 公開予定日 [日付]
- 次のアクション: Antigravity で image-prompt.md のプロンプトを使って画像生成
```
